\bigskip

{\bf Объектом исследования} являются задачи оптимального управления на бесконечном полуинтервале времени, обладающие магистральными свойствами.

{\bf Цель работы} --- применить методы управления по прогнозирующей модели для отслеживания магистралей в рассматриваемых задачах оптимального управления.

{\bf Методы исследования} --- методы управления по прогнозирующей модели, методы оптимального управления, методы оптимизации.

{\bf Результатом} работы является построение магистралей для задач оптимального управления, используя оптимизацию в реальном времени и метод управления по прогнозирующей модели. При решении задач управления с прогнозирующей моделью учитываются ограничения, накладываемые на состояние объекта и управление.

{\bf Областью} применения являются аналитические исследования и практическая оптимизация процессов в режиме реального времени при ограниченях на управляющие и управляемые переменные.
